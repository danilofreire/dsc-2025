# Show, Don't Tell: GitHub as Hidden Curriculum

**A talk for the Data Science Club at Emory University**  
**Date:** 02 December, 2025  
**Speaker:** [Danilo Freire](https://danilofreire.github.io), Visiting Assistant Professor, Department of Data and Decision Sciences

## Overview

This repository contains materials for a talk about how GitHub can help you navigate the hidden curriculum of the job market and improve your employability as a data scientist.

The presentation slides are available at <https://danilofreire.github.io/dsc-2025/dsc-2025.html>.

## What You'll Learn

This talk covers:

1.  Understanding some unwritten rules of job applications
2. A (very brief) introduction to GitHub workflows
3. How to structure and document your projects professionally
4. Using branches, pull requests, and issues to showcase your contributions
5. Creating your first portfolio website in under 5 minutes
6. What to do before sending your GitHub profile to recruiters
